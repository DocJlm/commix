# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-03-11

### Added
- Initial release
- AI-powered commit message generation
- Support for Conventional Commits format
- Gitmoji emoji support
- Multiple AI providers (OpenAI, Claude, Ollama)
- Interactive mode for message selection/editing
- Configuration management with YAML
- Git utilities integration
- Template system for custom commit formats

### Features
- Smart diff analysis
- Auto-detect commit type (feat, fix, docs, etc.)
- Batch commit mode for large changes
- Pre-commit hook support
- CI/CD integration examples

### Dependencies
- typer >= 0.9.0
- rich >= 13.0.0
- pyyaml >= 6.0
